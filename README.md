# Solaris Assets — Frontend
 
Prototipo de la interfaz de administración de expensas de inquilinos.
 
## Requisitos
 
- Node.js 18 o superior
- npm 9 o superior
 
## Instalación y uso
 
```bash
git clone <url-del-repo>
cd solaris-assets
npm install
npm run dev
```
 
Abrir http://localhost:5173 en el navegador.
